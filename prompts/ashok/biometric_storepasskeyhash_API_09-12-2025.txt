   - Sets passkey: `user.setPasskey(request.passkeyHash());`
   - Stores passkeyHash in `PASSKEY` column (TEXT type)
   - Enables biometric: `user.setBiometricEnabled(true);`
   - Sets `BIOMETRIC_ENABLED` column to `true`
   - Sets timestamp: `user.setPasskeyCreatedAt(LocalDateTime.now());`
   - Sets `PASSKEY_CREATED_AT` column to current timestamp
   - Saves entity: `userRepository.save(user);`
   - `save()` method:
     - If entity has ID: performs UPDATE SQL
     - If entity has no ID: performs INSERT SQL
     - In this case: UPDATE (user already exists)
     - SQL: `UPDATE MEDGO_USERS SET PASSKEY = ?, BIOMETRIC_ENABLED = ?, PASSKEY_CREATED_AT = ? WHERE id = ?`
   - Logs: `LOGGER.info("Passkey hash stored for member user: {} at {}", request.email(), LocalDateTime.now());`
   - Returns: `CommonResponse.success("Passkey hash stored successfully")`

4. NON_MEMBER User Lookup:
   - If MEMBER not found: `Optional<NonMemberUserModel> nonMemberUserOpt = nonMemberUserRepositry.findByEmailOrMobile(...)`

5. NON_MEMBER User - Passkey Storage:
   - Gets user: `NonMemberUserModel user = nonMemberUserOpt.get();`
   - Sets passkey: `user.setPasskey(request.passkeyHash());`
   - Sets timestamp: `user.setPasskeyCreatedAt(LocalDateTime.now());`
