9.2 Feign Client Layer - Detailed Explanation Prompt

ROLE: You are a Spring Boot developer explaining Feign client integration and automatic authentication.

TASK: Explain in detail what happens in the Feign client layer when calling the Medicard API for LOA download.

CONTEXT:
- Feign Client: `MedicardApiClient`
- Endpoint: `GET /mcap/mace-medgo/form/loa/v2`
- Base URL: Configured via `medicard.api.url` property
- Configuration: `FeignClientConfig` (automatic token injection)
- Response Type: `LoaDownloadResponseDTO` (typed response)

WHAT HAPPENS IN THIS LAYER:

1. Feign Client Interface:
   - `@FeignClient(name = "medicard-api", url = "${medicard.api.url}", configuration = FeignClientConfig.class)`
   - Method: `LoaDownloadResponseDTO downloadLoa(String systemOrigin, Long requestId, String originMemberCode);`
   - Mapping: `@GetMapping("/mcap/mace-medgo/form/loa/v2")`
   - Query Parameters: `@RequestParam` for all three parameters
   - Response Type: `LoaDownloadResponseDTO` (not Object - typed response)

2. Request Interceptor Execution:
   - `FeignClientConfig.combinedHeadersInterceptor()` intercepts request
   - Executes before HTTP request is sent

3. APIM Key Header Addition:
   - Checks: `if (loaApimKey != null && !loaApimKey.isEmpty())`
   - Adds header: `requestTemplate.header("Ocp-Apim-Subscription-Key", loaApimKey);`
   - APIM key from: `@Value("${loa.apim-key}")` property

4. Authentication Token Retrieval:
   - Calls: `String accessToken = tokenService.getAccessToken();`
   - `TokenService.getAccessToken()`:
     a. Checks cached token validity (with 5-minute buffer)
     b. If valid: Returns cached token
     c. If expired/invalid: Fetches new token from Medicard Auth API
     d. Caches new token with expiry time
     e. Returns access token

5. Authorization Header Addition:
   - Extracts token value (removes "Bearer " prefix if present)
   - Adds header: `requestTemplate.header("Authorization", "Bearer " + tokenValue);`
   - Token format: `Bearer <access_token>`

6. HTTP Request Execution:
   - Feign client sends HTTP GET request to Medicard API
   - URL: `{medicard.api.url}/mcap/mace-medgo/form/loa/v2`
   - Query Parameters:
     - `systemOrigin`: System origin (MCAP or MACE)
     - `requestId`: LOA request ID
     - `originMemberCode`: Member code
   - Example URL: `/mcap/mace-medgo/form/loa/v2?systemOrigin=MCAP&requestId=21909&originMemberCode=2621761`
   - Headers:
     - `Authorization: Bearer <token>`
     - `Ocp-Apim-Subscription-Key: <apim-key>`

7. Response Handling:
   - Receives response from Medicard API
   - Response format (JSON):
     ```json
     {
       "statusCode": "200",
       "statusName": "SUCCESS",
       "data": {
         "fileBase64": "JVBERi0xLjQKJeLjz9MKMSAwIG9iago8PC9UeXBlL0NhdGFsb2cvUGFnZXMgMiAwIFI+PgplbmRvYmoK...",
         "fileName": "LOA_21909.pdf",
         "createdOn": "2025-01-15T10:30:00",
         "createdBy": "System"
       }
     }
     ```
   - Deserializes JSON response to `LoaDownloadResponseDTO`
   - Jackson deserialization:
     - Maps `statusCode` to `String statusCode`
     - Maps `statusName` to `String statusName`
     - Maps `data` to nested `Data` object
     - Maps `fileBase64` to `String fileBase64` (Base64-encoded PDF)
     - Maps `fileName` to `String fileName`
     - Maps `createdOn` to `String createdOn`
     - Maps `createdBy` to `String createdBy`
   - Returns typed `LoaDownloadResponseDTO` to controller

8. Error Handling:
   - `FeignClientConfig.errorDecoder()` handles errors
   - 401 Unauthorized: Clears cached token, logs error
   - 403 Forbidden: Clears cached token, logs error
   - 404 Not Found: Request ID not found, returns 404 with error details
   - Other errors: Uses default error decoder

CLASSES AND METHODS INVOLVED:
- `MedicardApiClient` - Feign client interface
- `FeignClientConfig` - Feign configuration with interceptors
- `TokenService` - Token management service
- `LoaDownloadResponseDTO` - Response DTO class
- `RequestInterceptor` - Feign request interceptor
- `ErrorDecoder` - Feign error decoder
- Jackson `ObjectMapper` - JSON deserialization
